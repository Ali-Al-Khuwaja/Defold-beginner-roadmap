A road-map for learning Defold

Introduction

What is Defold ?

Defold is a 3D game engine with focus on 2D, it's completely free to use and truly cross-platform engine. There are no licensing fees and no royalties. The source code is available on GitHub. Defold is known for its ease of use and it is praised for its technical documentation and friendly community of developers.

For more information click here.

What makes Defold special ?

it's visual editor, drag-and-drop functionality, and beginner-friendly scripting language.

more info here.

Who is this book road-map for ?

This bookroad-map is for complete beginners who may have no programming or game development experience and want to learn Defold.

How to use this book road-map ?

The book road-map has chapters that build on each other,we will make some mini-projects and finish it with a final game.

The book road-map is written by a single person who is also learning just like you and the primary goal is to achieve the Minimum Viable Product (MVP) first, and then focus on refining it, so throughout the whole book road-map you will be given links to some lesson to read because it's already exits.

Setting up your workspace:

let's start setting up your workspace, first you should install Defold if you haven't done it already, then read this introduction to Defolds editor.
